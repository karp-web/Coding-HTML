HTML

h1~h6는 제목이다. (1이 제일 큰 제목)
strong은 굵은글씨, u는 underline 밑줄(u는 잘 사용하지 않음)
br은 줄바꿈, p는 문단
li는 목록, ul은 그냥 목록과 내용 구분, ol은 목록들 앞에 자동으로 순번(ol은 잘 사용하지 않음)
title은 웹의 제목
meta charset은 인식 안될 때, 본문 내용이 무엇으로 되어있는지 컴퓨터한테 말해주는 것
코딩의 구조 순서 : !DOCTYPE html - html - head - body
a는 링크, href는 하이퍼링크 참조 a안의 title은 링크의 제목
